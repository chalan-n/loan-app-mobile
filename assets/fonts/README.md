# วิธีดาวน์โหลดฟอนต์ Kanit

## ขั้นตอนที่ 1: ดาวน์โหลดจาก Google Fonts
1. เปิดเว็บ: https://fonts.google.com/specimen/Kanit
2. คลิกปุ่ม "Download family" ด้านขวาบน
3. จะได้ไฟล์ ZIP ชื่อ Kanit-v15.zip

## ขั้นตอนที่ 2: แตกไฟล์และคัดลอก
1. แตกไฟล์ Kanit-v15.zip
2. คัดลอกไฟล์ต่อไปนี้ไปที่ assets\fonts\:
   - Kanit-Regular.ttf
   - Kanit-Medium.ttf  
   - Kanit-SemiBold.ttf
   - Kanit-Bold.ttf

## ขั้นตอนที่ 3: รันคำสั่ง
flutter pub get
flutter run -d R52N517MTTR

## ตรวจสอบไฟล์
หลังจากคัดลอกแล้วต้องมีไฟล์:
- assets/fonts/Kanit-Regular.ttf
- assets/fonts/Kanit-Medium.ttf
- assets/fonts/Kanit-SemiBold.ttf
- assets/fonts/Kanit-Bold.ttf
